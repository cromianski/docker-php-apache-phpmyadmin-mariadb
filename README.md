# docker-php-apache-phpmyadmin-mariadb

Default project stucture to build an application with:
- Docker
- PHP
- Apache
- PHPMyAdmin
- MariaDB

## Build (first time only)

``` bash
  docker-compose up -d --build
```

## Start

``` bash
  docker-compose up
```

## Shutdown

``` bash
  docker-compose down
```

## Delete containers

``` bash
  docker system prune -a
```
